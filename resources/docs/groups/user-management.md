# User management

APIs for managing users

## Log in .


This endpoint lets you login with credentials

> Example request:

```bash
curl -X POST \
    "http://localhost:8005/api/login?email=maiores&password=provident" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json"
```

```javascript
const url = new URL(
    "http://localhost:8005/api/login"
);

let params = {
    "email": "maiores",
    "password": "provident",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};


fetch(url, {
    method: "POST",
    headers,
}).then(response => response.json());
```

## Log out .

<small class="badge badge-darkred">requires authentication</small>

This endpoint lets you log out

> Example request:

```bash
curl -X GET \
    -G "http://localhost:8005/api/logout" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json"
```

```javascript
const url = new URL(
    "http://localhost:8005/api/logout"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};


fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
```


> Example response (200):

```json

{
'User logged out'
}
```
